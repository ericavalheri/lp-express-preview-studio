# LP Express Studio Standalone

Aplicação standalone do LP Express Studio, fora do WordPress.

- Front-end: React + Vite
- Domínio principal atual de staging: studio-app-lp.agenciacavalheri.com
- Studio começa vazio: a LP só nasce após briefing ou comandos da Zaya
- Studio voltou para visual escuro/premium no painel
- Zaya local agora infere segmento em pedidos como oficina mecânica, preenche briefing, imagem e modelo Serviço Impacto
- Serviço Impacto expandido: hero, prova, serviços, processo, showcase, Google Maps, FAQ, CTA e rodapé
- Campos de Endereço e Google Maps/iframe voltaram ao briefing
- `view=preview` abre a LP final em tela cheia, sem painel do Studio

Não salvar credenciais neste repositório.
